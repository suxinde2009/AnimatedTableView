# AnimatedTableView
	
	AnimatedTableView is a drop-in class of UIView,
	which lets you add custom animations to any UITableView.
	

<br/>
![Demo](./Docs/demo.gif)

# How to use

   Check out the `AnimationTableViewDemo` target of `AnimatedTableView.xcworkspace`.
